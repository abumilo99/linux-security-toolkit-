# Linux Security Toolkit

A Bash-based Linux security toolkit that simulates real-world
**offensive reconnaissance** and **defensive system monitoring**
used by cybersecurity analysts and SOC teams.

##  Features

### Reconnaissance (Ethical Simulation)
 Host discovery
 Port availability checks

### Monitoring & Detection
 Failed login detection
 Brute-force attempt alerts

### System Audit
SUID file detection
 World-writable file checks

## Disclaimer
This project is for **educational and defensive purposes only**.
All scripts are designed to be used in **authorized lab environments**.

Bash
 Linux (Ubuntu)
- Core GNU tools
